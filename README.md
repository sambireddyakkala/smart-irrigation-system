# Smart Irrigation System

## Overview
This project automates water supply for plants using a soil moisture sensor and Arduino.

## Components Used
- Arduino UNO
- Soil Moisture Sensor
- Relay Module
- Water Pump
- Jumper Wires

## Features
- Automatic water pumping
- Soil moisture monitoring
- Water saving system

## Technologies Used
- Embedded C
- Arduino IDE

## Applications
- Agriculture
- Smart farming
- Water conservation
